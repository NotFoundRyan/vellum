## 功能特色

- **实时预览** - 左侧编辑，右侧即时渲染，所见即所得
- **多主题支持** - 内置 7 款精美主题，一键切换
- **一键复制** - 渲染后的富文本可直接粘贴到微信公众号编辑器
- **代码高亮** - 支持多种编程语言语法高亮
- **主题扩展** - 支持导入自定义主题 JSON 文件
- **文件操作** - 支持新建、打开、保存 Markdown 文件
- **导出 HTML** - 可导出渲染后的 HTML 文件

## 排版规范

素笺遵循微信公众号最佳排版实践：

### 字号层级

| 元素    | 字号   | 说明         |
| ----- | ---- | ---------- |
| 大标题   | 22px | 加粗，居中，最大字重 |
| 小标题   | 17px | 加粗，可配左边框装饰 |
| 正文    | 15px | 黄金字号，阅读最舒适 |
| 引用/注释 | 14px | 辅助信息       |

### 颜色规范

- **正文色**：`#3f3f3f`（高级深灰，非纯黑）
- **小标题**：比正文深一个度或使用品牌色
- **辅助色**：`#888888` \~ `#a3a3a3`

### 间距规范

- **行间距**：1.75（黄金值）
- **字间距**：1px（精致感）
- **段落间距**：15px（呼吸感）

## 自定义主题

### 主题文件结构

主题采用 JSON 格式，存放在 `src/themes/` 目录：

```json
{
    "name": "主题名称",
    "dotColor": "#品牌色",
    "brandColor": "#品牌色",
    "h1": "font-size:22px;font-weight:900;color:#颜色;...",
    "h2": "font-size:17px;font-weight:700;...",
    "p": "font-size:15px;color:#3f3f3f;line-height:1.75;...",
    ...
}
```

### 创建新主题

1. 复制 `theme-template.json` 模板文件
2. 修改 `name` 和颜色配置
3. 通过菜单「主题 → 导入主题」导入

### 导入主题

1. 点击菜单栏「主题」→「导入主题...」
2. 选择 JSON 格式的主题文件
3. 导入成功后自动出现在主题列表

### 打开主题目录

点击菜单「主题」→「打开主题目录」可直接访问主题文件夹，方便批量管理。

## 特殊语法

### 金句卡片

短引用（少于 100 字）会自动渲染为居中的金句卡片样式：

```markdown
> 好的排版不是装饰，而是对读者的尊重。
```

### 图片说明

图片后紧跟的斜体文字会自动转为图片说明：

```markdown
![图片](url)
*这是图片说明*
```

## 常见问题

### 复制到微信后样式丢失？

微信公众号编辑器会过滤部分样式。素笺已针对微信兼容性优化：

- 所有样式使用内联 `style` 属性
- 代码高亮颜色直接写入元素
- 不依赖外部 CSS 文件

### 如何修改默认主题？

主题按文件名字母顺序加载，第一个主题为默认主题。可将喜欢的主题文件重命名为 `aaa-主题名.json`。

### 支持哪些 Markdown 语法？

支持标准 GFM（GitHub Flavored Markdown）语法，包括：

- 标题、段落、引用
- 加粗、斜体、删除线
- 有序/无序列表
- 代码块、行内代码
- 表格、链接、图片
- 水平分割线
